# Sriram Stories | Production Suite

A private, professional-grade web environment for filmmakers and scriptwriters.

## 🕹 Functionality
- **Home:** Minimalist entry point.
- **Studio Mode:** Full-screen white writing environment.
- **Tools:** Bold, Italic, H1, and Image Attachment features included in the Studio.
- **Review Portal:** Direct file upload system for script verification.

## 🚀 Deployment
1. Upload `index.html`, `style.css`, and `script.js` to a GitHub repository.
2. Settings > Pages > Select "Main" branch > Save.
